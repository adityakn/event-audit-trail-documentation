Getting Started with Microservices

Microservices are a style of software architecture that involves delivering systems as a set of very small, granular, independent collaborating services. The Predix platform has a Cloud Foundry-based microservice usage and delivery architecture. Services available in Cloud Foundry are visible in the Cloud Foundry marketplace. Services available in Cloud Foundry enable you to provision service instances. You can then integrate the service instances with your applications that reside in Cloud Foundry.

For more information on services and service instances in Cloud Foundry, see http://docs.cloudfoundry.org/devguide/services/.

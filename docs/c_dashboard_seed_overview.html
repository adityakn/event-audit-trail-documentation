Dashboard Seed Overview

The Predix UI platform includes a UX framework that combine the following elements:
  • Components — User-interface element that allows you to add features or access services following consistent UX practices, standards and specification.
  • Cards — Container that resides within the content area of the user interface. You can create interactions between components within an individual card, as well as interactions with other cards based on user context.
  • Decks — Container that contains cards and components arranged in a specified order.
  • Views — Visual summary of the information displayed in a web application. It consists of one or more decks that an application user can select at runtime to quickly digest information and take appropriate action (see About the View Service for more information).

The Dashboard Seed is a prototype that demonstrates the use of the Predix UI platform (http://predixdev.github.io/predix-ui/) to build a basic web application than runs on top of Predix services and data.

The Dashboard Seed provides you with the following features and services:
  • Cloud Foundry Deployment Support
  • Contextual Dashboard
  • Time Series Chart
  • Vertical Navigation
  • Contextual Browser
  • Cards
  • Decks
  • Data Binding
  • View Switching

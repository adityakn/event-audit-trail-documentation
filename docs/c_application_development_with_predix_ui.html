Polymer Library

As a developer, you should understand the Polymer library before using the Predix UI platform. Familiar HTML elements such as button, form, and table often require HTML, CSS, and scripting support.

The Polymer library extends HTML to facilitate sophisticated user-interface presentation in your web applications. The Predix UI repositories are implemented as custom elements using the Polymer library (see Predix UI Repository Overview for more information.).

The Polymer library also provides a set of features for creating custom, reusable elements for web applications. It supports registration, callbacks, DOM templates, data binding, and property observation. You can use these features to efficiently build complex, interactive applications across web browsers (see Supported Browsers for Web Applications for more information).

See the following resources for more information about the Polymer library:
  • Polymer Library Overview
  • Polycasts
  • Build Your First Polymer Element
  • Polymer App Toolbox

Developer Tools

As a developer, you should understand the following open-source tools before using the Predix UI platform:
  • GitHub code repository
  • Node.js runtime environment
  • Bower package manager
  • Gulp build system
  • CSS framework
  • SASS (Syntactically Awesome Stylesheets) language
GitHub Code Repository

As a developer, you use GitHub as a source code repository and version control system. The Predix UI platform is modular by design, allowing you to add only the features and functions that you need from the individual repositories available in the PredixDev public GitHub organization..

See the following resources for more information about the GitHub code repository:
  • GitHub Overview
  • GitHub Documentation
Node.js Runtime Environment

As a developer, you use Node.js as a cross-platform runtime environment to build web applications. Node.js includes the npm utility to manage node modules such as the Bower package manager and the Gulp build system.

See the following resources for more information about the Node.js runtime environment:
  • Node.js Overview
  • Node.js Documentation
Bower Package Manager

As a developer, you use the Bower package manager to install the correct package versions needed to configure a web application (such as HTML, CSS, or JavaScript files) and their dependencies. Bower fetches and installs the required packages, then records the package dependencies in a bower.json manifest file.

See the following resources for more information about the Bower package manager:
  • Bower Overview
  • Bower Documentation
Gulp Build System

As a developer, you use the Gulp build system to start a web server that runs tasks and serves files to build a web application. This web server can be accessed by entering http://localhost:9000/ in a web browser .

See the following resources for more information about the Gulp build tool:
  • Gulp Overview
  • Gulp Documentation
CSS Framework

The Predix UI design system is implemented using the incuit.css CSS framework (see Predix UI Design System Overview for more information). As a developer, you can use this library of standardized code (HTML, CSS, and JavaScript) as a basis to create your web application. It provides the following benefits:

  • Web applications that follow consistent UX practices, standards, and specifications.  
  • Modular architecture that provides the flexibility to change and configure existing web applications.
  • Responsive web design that supports laptop, tablet, phone, or large-screen devices.
See the following resources for more information about CSS frameworks:
  • What is a CSS Framework?
  • Inuit.css v5.0 The Object Oriented Framework
SASS Language

The Predix UI design system libraries use the SASS stylesheet language (see Predix UI Design System Libraries for more information). SASS extends CSS by providing several mechanisms available in object-oriented programming languages:

  • Preprocessing
  • Variables
  • Nesting
  • Import
  • Mixins
  • Inheritance
  • Operators
See the following resources for more information about the SASS language.

  • SASS Tutorials
  • SASS Documentation

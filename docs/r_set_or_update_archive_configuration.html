Set or Update Archive Configuration

This service sets or updates the archive configuration.

METHOD POST
URI POST archive/tenant/{tenant_uuid}/configuration
Sample Request 
{
  "maximumNumberOfEvents": 100,
  "maximumNumberOfStoredEventsDays": 30,
  "stagedEventThreshold": 50
}

Request Body JSON Parameters

Parameter Description Required?
maximumNumberOfEvents Maximum number of events to retain (Default = -1). No
maximumNumberOfStoredEvents Days Maximum numbers of days of events to retain (Default = -1). No
stagedEventThreshold Maximum staged events before events are permanently archived (Default = -1). No

Setting Up Required Services for Predix UI

To use the Predix UI platform, you must first set up the following services:

  1. Set up the Predix developer environment. For more information, see Creating the Predix Developer Environment.
  2. Install the latest version of the Node.js runtime environment for your operating system. For more information, see https://nodejs.org/en/download/.
  3. Modify the file permissions for the Node.js npm package-management utility. For more information, see https://docs.npmjs.com/getting-started/fixing-npm-permissions.
  4. Install the bower browser package-manager utility. For more information, see https://www.npmjs.com/package/bower. 
    npm install bower -g

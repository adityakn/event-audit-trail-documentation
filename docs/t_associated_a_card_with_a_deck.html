Associating a Card with a Deck

You need a REST client (such as Postman) that can execute API requests to use the View service. See View Service Database Operations for more information.

For more information about this API, see the API Documentation.

Associate one or more cards to a deck in the View service database and retrieve all cards.
  1. Associate a card with a deck once they are created in the View service database. For example:
    /*
Associate Card 1,2 to Deck 1
[POST] /api/decks/1/cards/add
Request body 
*/
["1", "2"]
  2. Retrieve all cards in a deck from the database in JSON format. For example
    /*
Get all cards in deck 1
[GET] /api/decks/1/cards
Response Body:
*/
    [
        {
            "id": 1,
            "title": "San Ramon Temperature",
            "slug": "my-temperature-card",
            "attributes": {
                "postal": "94582,
                "unit": "F"
            }
        },

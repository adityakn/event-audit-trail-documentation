Get Archive by UUID

This service returns archives by UUID.

METHOD GET
URI GET archive/tenant/{tenant_uuid}/archives/{uuid}
Sample Request 
[
    {
      "context": "user test",
      "tag": "tag1",
      "classification": 0,
      "data": "{\"name\": \"event1\",\"type\": \"request\",\"params\": \"test\"}"
    }
  ]

Path Parameters

Parameter Description Required?
tenant_uuid UUID of the tenant. Primary event grouping. Yes
uuid UUID of the archive. Yes

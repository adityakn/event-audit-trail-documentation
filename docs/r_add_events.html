Add Events

This service creates or adds events.

METHOD POST
URI POST tenants/{tenant_uuid}/events
Sample Request 
[
    {
      "context": "user test",
      "tag": "tag1",
      "classification": 0,
      "data": "{\"name\": \"event1\",\"type\": \"request\",\"params\": \"test\"}"
    }
  ]

Request Body JSON Parameters

Parameter Description Required?
tenantUuid UUID of the tenant. This is the primary method of tag events. Yes
context User-defined context. No
tag User-defined tag. No
classification User-defined classification. No
timestamp If no timestamp is given, this will be set to the current time. No
data User-defined JSON data field. No

Creating a Deck

You need a REST client (such as Postman) that can execute API requests to use the View service. See View Service Database Operations for more information.

For more information about this API, see the API Documentation.

A deck is a container that determines the layout of cards.
  1. Create a deck instance in the View service database. For example:
    <!-- a deck with 2 cards -->
<px-deck id="deck-1">
    <my-temperature-card id="card-1" title="San Ramon Temperature"postal="94582" unit="F"></my-temperature-card>
    <my-temperature-card id="card-3" title="Walnut Creek Temperature" postal="94595" unit="F"></my-temperature-card>
</px-deck>
  2. Retrieve a deck instance in JSON format from the database. For example:
    /*
Create deck in view service
[POST] /api/decks
Request Body:
*/
{
    "title": "Contra Costa Deck"
}

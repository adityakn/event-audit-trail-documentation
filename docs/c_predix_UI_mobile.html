Predix UI Mobile Components

Predix UI includes a set of components in the PredixDev public GitHub organization for specific use in mobile devices (such as tablets or phones).

As a developer, you can use the code in the following Predix UI mobile component repositories to produce these interactions.

Table 1. Predix UI Mobile Components
Repository Description Documentation
px-action-sheet User interface element slides up from the bottom of the screen to reveal more content. API Reference
px-drawer Navigation drawer that can slide in from the left, right, top, or bottom. API Reference
px-table-view User interface element that displays list data. API Reference

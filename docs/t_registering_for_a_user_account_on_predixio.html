Registering for a Predix Platform User Account

When you register for an account, an org and space is created for you in Cloud Foundry. You can register as an individual user or as an enterprise user. Individual users can add additional spaces to an org assigned to them. Enterprise users can add additional users and spaces to the org assigned to them using the Predix.io user interface.

  1. To register as an individual user, go to http://predix.io and click Free Trial.
    If you are an enterprise user, contact Predix Customer Support at gedigital@ge.com for help setting up your account.
  2. Enter your information on the form, and click Continue Registration.
  3. Enter your physical business or personal address, and click Verify Location.

    When you register, the value of Cloud Foundry domain (<API_Endpoint>) is sent to you in the Predix sign-up email. Depending on your Predix.io registration, the value of <API_Endpoint> is one of the following:

    • Predix Basic

      https://api.system.aws-usw02-pr.ice.predix.io

    • Predix Select

      https://api.system.asv-pr.ice.predix.io

    • Predix Japan

      https://api.system.aws-jp01-pr.ice.predix.io

    • Predix UK

      https://api.system.dc-uk01-pr.ice.predix.io

  4. Sign in to Predix.io using the email and password you provided during registration.
    When you register for a Predix account, you are given a Cloud Foundry org and space. An org is a development account that a single developer or a group of collaborators can use. Each org contains a space, which provides access to a shared location for application development, deployment, and maintenance.

    The org and space(s) assigned to you are listed on the Console page. The orgs and spaces are assigned as follows:

    User Type Org Name Default Space Name Description
    Individual Email address specified by the user during registration. dev An individual account provides you with one org and one space within that org. If you have an individual account, you cannot add users to your org.
    Enterprise Company name specified by the user during registration. dev An Enterprise account provides you with one org and one or more spaces. Enterprise accounts allow you to create teams by inviting team members to be part of the org, and then adding them to the various spaces.

Setting Up Required Services to Install the Dashboard Seed

Follow these steps to set up the services required to install the Dashboard Seed:

  1. Set up the Predix developer environment. For more information, see Creating the Predix Developer Environment.
  2. Install the latest version of the Node.js runtime environment for your operating system to set up the developer tools needed to deploy a web application to Cloud Foundry. For more information, see https://nodejs.org/en/download/.
  3. Modify the file permissions for the Node.js npm package-management utility. For more information, see https://docs.npmjs.com/getting-started/fixing-npm-permissions.
  4. Install the grunt and bower package-management utilities: 
    npm install grunt grunt-cli bower -g

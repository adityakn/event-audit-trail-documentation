Deploying a Predix Hello World Web Application

If you are a new user, consider following these steps to deploy a simple Predix Hello World web application.

  1. Use the Cloud Foundry CLI to log into Cloud Foundry. 
    cf login -a <API_Endpoint>

    The value of <API_Endpoint> is available in the Predix welcome email that you get when you register for an Predix.io user account. Depending on your Predix.io registration, the value of <API_Endpoint> is one of the following:

    • Predix Basic

      https://api.system.aws-usw02-pr.ice.predix.io

    • Predix Select

      https://api.system.asv-pr.ice.predix.io

    • Predix Japan

      https://api.system.aws-jp01-pr.ice.predix.io

    • Predix UK

      https://api.system.dc-uk01-pr.ice.predix.io

    For example,

    cf login -a https://api.system.aws-usw02-pr.ice.predix.io
    The command produces the following output: 
    Email> <your_predix_login> 
Password> <your_predix_password> 
Authenticating...OK

  2. Clone your copy of the Predix Hello World web application and start a Predix Hello World project. For example:
    git clone https://github.com/PredixDev/Predix-HelloWorld-WebApp
    The command produces the following output: 
    Cloning into
    'Predix-HelloWorld-WebApp'...

    done.

    reused 7 (delta 0), pack-reused 0

  3. Edit the manifest.yml file in the Predix-HelloWorld-WebApp directory to update the name of the web application to Predix-HelloWorld-WebApp-<YourAppName>. For example:
    #
#  A basic hello world application
#  This application, and steps to deploy are described in full in the getting started guide on predix.io
#  Step 1 is to change your application name, save, exit, and cf push
#  Step 2 - After you have your application running you can bind to the services below as described on predix.io
 
applications:
  - name:
      Predix-HelloWorld-WebApp-<YourAppName> # Step 1: Change your application
  name
    buildpack: predix_openresty_buildpack
    #path: dist
    memory: 64M
    stack: cflinuxfs2

    # Step 2: Bind services and configure environment variables as described on predix.io
    #services:
    #        - my_redis_instance # change this to your redis service instance name
    #        - my_viewsvc_instance # change this to your view service instance name
    #env:
    #  UAA_SERVER_URL: <my-UAA-URL>
    #  REDIS: redis-<num>
  4. Make sure that you are in the Predix-HelloWorld-WebApp directory. Use the Cloud Foundry CLI to deploy your Predix web application to Cloud Foundry. For example:
    cf push

    Your basic Predix Hello World web application is deployed to Cloud Foundry.

  5. See your Predix web application run:
    1. Use the Cloud Foundry CLI to see your web application in Cloud Foundry:
      cf apps
    2. Enter the web application URL into your web browser. For example:
       https://Predix-HelloWorld-WebApp-<YourAppName>.run.aws-usw02-pr.ice.predix.io
    Your basic Predix Hello World web application displays in a web browser.

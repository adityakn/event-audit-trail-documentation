Predix UI Design System Libraries

The Predix UI design system includes a set of design system libraries in the PredixDev public GitHub organization to create intricate user interactions in a web application..

Each Predix UI design system library uses SASS language (.scss) files to extend CSS style sheets.

Each Predix UI design system library includes a bower.json manifest file that lists the other repositories required for successful compilation of the SASS files contained in the parent repository. This is displayed as a series of @import declarations. The bower task-runner utility uses the bower.json file file to import the correct version of other required repositories.

As a developer, you can use the code in the following Predix UI design system library repositories to produce these interactions.

Tagging a Deck

You need a REST client (such as Postman) that can execute API requests to use the View service. See View Service Database Operations for more information.

For more information about this API, see the API Documentation.

You can tag a card or deck for look up by category in the View service database.
  1. Tag a deck to group by multiple values. For example:
    *
Tag a deck by multiple tag values
[POST] /api/decks/1/tags
Request Body:
*/
[
{"value": "contra costa"},
{"value": "california"},
{"value": "us"}
]
  2. Look up a deck by tag. For example:
    /*
Get a deck by tag/category lookup
[GET] /api/decks/tags?values=us
Response Body:
*/
[
    {
        "id": 1,
        "title": "Contra Costa Deck"
        "tags": [
            {"value": "contra costa"},
            {"value": "california"},
            {"value": "us"}
        ]
    }
]

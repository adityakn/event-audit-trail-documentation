Get Events by Tenant with Filters

This service returns a list of tenants using filters.

METHOD GET
URI GET /tenants/{tenant_uuid}/events?context=&tag=&classification=&start_date=&end_date=
Sample Request 
[
    {
      "context": "user test",
      "tag": "tag1",
      "classification": 0,
      "data": "{\"name\": \"event1\",\"type\": \"request\",\"params\": \"test\"}"
    }
  ]

Path Parameters

Parameter Description Required?
tenant_uuid UUID of the tenant. Yes

Query Parameters

Filters Description Required?
context Events by context. Yes
tag Events by tag. Yes
classification Filters events by classification. No
start_date Filters events by start_date in the format: yyyy-M-d hh:mm:ss, inclusive. If a start_date is supplied, an end_date must also be supplied. No
end_date filters events by end_date in the format: yyyy-M-d hh:mm:ss, exclusive. If an end_date is supplied, a start_date must also be supplied. No

Installing the Predix UI Design System Starter Kit

  • Set up the required services for Predix UI. See Setting Up Required Services for Predix UI for more information.
  • Locate the application project directory <app_project> where the px-starter-kit-design files will be installed
  • Install the grunt task-runner utility for npm. See https://www.npmjs.com/package/grunt for more information.
  • Install the autoprefixer utility that parses CSS and adds vendor prefixes to CSS rules. See https://github.com/postcss/autoprefixer for more information
Note: See https://github.com/PredixDev/px-getting-started for more information on installing the Predix UI design system files stored on the PredixDev public GitHub organization.

Follow these steps to install the Predix UI design system starter kit.

  1. Change to the application project folder:
    cd <app_project>
  2. Install the Predix UI design system starter kit files: 
    bower --save install https://github.com/predixdev/px-starter-kit-design.git

    The Predix UI design system starter kit files are installed in the bower_components directory.

See Using the Predix UI Design System Starter Kit for specific information about using the Predix UI design system starter kit.
